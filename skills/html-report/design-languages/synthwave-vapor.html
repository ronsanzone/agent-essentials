01

Personality

Vapor is the softer cousin of Outrun. Where Outrun shouts in saturated magenta, Vapor whispers in pink-soft #FFB3D9. The base is a deep dark-purple #1a0d2e rather than near-black, and the body carries a permanent low-opacity CRT scanline overlay — barely visible, but it sets the mood. Display is VT323, the pixel monospace from 1980s terminals, used only for the hero h1 and section h2. Body is Space Grotesk, a slightly quirky geometric sans that pairs with VT323 without competing.

Vapor is what synthwave sounds like at 3 a.m., when the cassette tape has stretched and the colors have washed out.

Pink-soft leads. Cyan-soft is the secondary — eyebrows, pills, marginalia counter, the italic-em in the hero. Peach #FFCBA4 is the warn hue, mint #B5EAD7 the success. Nothing is fully saturated; nothing carries a hot neon glow. Cards have a soft pink wash for shadow, never a sharp edge.

02

When to pick this

Pick this when

Retrospectives on archival content. Music criticism with a wistful tone. Features on obsolete formats — cassette tapes, VHS, dial-up. Anywhere a softer, dreamier register is right and pure Outrun would be too loud.

Don't pick this when

The report needs to convey urgency or rigor. Audience is reading on a glare-prone screen (the scanline overlay can fight low-quality displays). Content is a 20+ minute technical reference — VT323's pixel forms at h2 are charming, not analytical.

Voice rules

  • VT323 only appears as display (h1, h2) or as mono (code, eyebrow, pills, num, th). Never as body.
  • The <em> inside h1 is cyan-soft. The italic-em is not italic shape-wise (VT323 has no italic) — it's a color shift.
  • Pull quotes use Space Grotesk italic 300, pink-soft, with a subtle CRT scanline background.
  • The eyebrow dot is a square, not a circle. The chunky square is part of the 90s-interface tone.
  • Never apply text-shadow glow to display text in Vapor. That's an Outrun-only move; Vapor stays soft.
  • Pills have beveled borders (lighter top/left, darker bottom/right) for the chunky 90s feel.

05

Anti-patterns

  • No VT323 for body text. Even committed retro types will give up at paragraph length.
  • No saturated magenta, electric cyan, or laser yellow. Those belong to Outrun.
  • No text-shadow glow. Vapor is a soft-edge language.
  • No pure black #000. The brand black is #1a0d2e with a violet bias.
  • No sharp 90° corners on cards. Vapor cards always round (12–20px).
  • No scanline overlay outside the body (don't double-layer it on cards or callouts).
  • No emoji in section headings.
  • No five-color rainbows. Pink-soft + cyan-soft + purple-soft are the load-bearing trio.

07

Diagram philosophy

Diagrams in Vapor look like they're being read off a CRT through a layer of pink fog. Every figure carries a scanline pattern overlay (1px horizontal stripes at low opacity, repeating every 3px) baked into the SVG, soft pastel pink and cyan fills, and a blur-halo behind every box. Labels are VT323 — the pixel terminal face — and arrows are dashed pink-soft with their own glow. Square decorative dots replace circles. Nothing is loud; the page hums.

SVG typography

Vapor diagrams are typeset entirely in VT323 — headlines and small labels alike — to commit to the pixel terminal aesthetic. Space Grotesk italic survives only for inline notes where ascender clarity matters.

.sv-txt   { font: 22px "VT323", monospace; fill: #f5e8ff; letter-spacing: 0.04em; }
.sv-small { font: 16px "VT323", monospace; fill: #d6c5ee; letter-spacing: 0.04em; }

Vapor extensions for annotated sequences:

.sv2-note  { font: italic 12px "Space Grotesk", sans-serif; fill: #a596c5; }
.sv2-step  { font: 18px "VT323", monospace; fill: #A0F0F0; letter-spacing: 0.08em; }
.sv2-alert { font: 18px "VT323", monospace; fill: #FFCBA4; letter-spacing: 0.10em; }

Arrow color

Default arrows are pink-soft, 1.6px, dashed 4 2, with an arrow-glow filter (matches the CRT softness). Return paths use arrow-muted (purple, no glow, tighter dash). Async dispatch uses a longer dash 6 3 and keeps the glow.
