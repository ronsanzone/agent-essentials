[01] PERSONALITY

Personality

CYBER 2077 is the HUD. Near-black backgrounds, corpo-yellow primary, cyan secondary, magenta where things go wrong. Rajdhani at heavy weights and tight tracking carries the display register; Share Tech Mono carries every coordinate, timestamp, and section pointer. Sharp corners. No gradients masquerading as warmth. The document should feel like it was rendered onto a visor inside a moving vehicle.

Two reports in this language should read like two intercepted broadcasts from the same network — same channel, different traffic.

The yellow is load-bearing. #FEE801 is the 2077 signature — replace it with amber or gold and the document loses its edge. The cyan is the operational counterpart: tooling output, ambient telemetry, supporting metadata. Magenta is reserved for failure paths and anything the operator needs to address immediately. Three colors. No fourth.

[02] DEPLOYMENT

When to deploy this

DEPLOY WHEN

Security audits with a confrontational tone. API references that document an edgy platform. Internal red-team reports where the visual edge supports the content. Game/runtime/protocol docs where "corpo HUD" is the right voice.

DO NOT DEPLOY WHEN

The audience is executive and non-technical — the all-caps register reads as shouting. The report is sequential long-form prose where reading fatigue matters — pick editorial-parchment. The content is celebratory or warm.

Voice rules

  • H1 and H2 are uppercase. H3 and H4 also uppercase but lighter weight. Body sentence-case.
  • The em inside h1 uses var(--yellow) at weight 500 — never italic.
  • Section numbers look like terminal coordinates: [01] LABEL.
  • Pills prefix with //; eyebrow prefixes with //; captions prefix with //. Slash-slash is this language's syntax for "machine-emitted label."
  • No italics anywhere. No serifs anywhere. No rounded corners (except rx on diagram nodes, kept under rx="2").

[05] ANTI-PATTERNS

Anti-patterns

  • No serif typefaces. No italic. No "warm" anything.
  • No purple/violet gradients. This is not synthwave. This is operational, not nostalgic.
  • No rounded corners over 2px. Buttons are blocks, cards are blocks, pills are blocks.
  • No emoji. No drop shadows softer than a 4px glow on the primary accent.
  • No light themes — this language is dark-only.
  • No paper grain, no parchment texture, no "soft" textile-mimicking patterns.
  • No bright photography. If you have a hero image, it should be UI-mimicking SVG.
  • No additional accent colors beyond the locked triad + neutrals.

4. Callout .callout

NOTE Default callouts use a yellow left border and a 6% yellow fill. The strong label is mono uppercase and color-matched.

CONFIRMED Success callouts use cyan. Use only when something has actually completed or passed verification — never as a generic "info" register.

[07] DIAGRAMS

Diagram philosophy

Diagrams are HUD elements. Sharp 2px corner radius max. Backgrounds match --bg-card so the diagram reads as a screen embedded in another screen. Text inside diagrams uses Rajdhani (matching body); coordinates and step markers use Share Tech Mono.

Alert region pattern

When a sequence has a failure leg, wrap the affected messages in a magenta-tinted dashed rectangle. Add an inline alert pill in saturated magenta with mono ALL-CAPS text explaining the consequence. The pattern is visible in the sequence diagram above between steps [03] and [04] — the magenta lands hard because that's the whole point. The retry pill is not "warn-soft" muted — it is the most saturated element on the page.

LOCKEDRegion uses rgba(237,30,121,0.08) + dashed magenta border. Inline alert pill stacks a magenta tick block, mono uppercase magenta label, and a Rajdhani neutral detail.
FREERegion size, alert pill copy, whether tick block is rectangle or square dot.

Arrow color

Default arrows: var(--yellow). 2px stroke. Async arrows: var(--cyan) dashed. Return paths use the muted border color. Warn arrows: var(--magenta), 2.5px stroke — the failure path is always heavier than the success path.

Caption

Mono uppercase, var(--text-dim), 12px, centered under the diagram with // prefix. See the captions on the figures above.

Two reports in this language should feel like two intercepted broadcasts from the same network. If they feel xeroxed, the report author leaned too hard on the locked elements. If they feel like different brands, someone reached for unauthorized accents.

Yellow on near-black is the channel. Cyan is the carrier. Magenta is the alarm. Anything else is interference.
