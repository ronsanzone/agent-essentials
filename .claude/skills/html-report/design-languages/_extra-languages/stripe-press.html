Design Language

Stripe Press, set in restrained editorial

A premium fintech register — every element earns its place. For technical product announcements, API launches, and B2B reports that want the trust signal of a Stripe press page without the marketing flourish.

Status: shipped Audience: external v1
LockedInter h1 700/800 at clamp(48px,6.5vw,88px), purple italic <em>, navy eyebrow text, purple eyebrow dot, white hero on #F6F9FC page.
FreePadding, lede max-width, hero meta as .meta-row key/values or .meta pill chips.
01

Personality

Stripe Press is the canonical Stripe press-page register: airy, restrained, cool, and crisp. Off-white #F6F9FC page background; white cards; deep navy #0A2540 for display ink and eyebrows; the recognizable Stripe purple #635BFF as the lead accent — links, italic-em in titles, active TOC, primary diagram nodes.

If a Stripe blog announcement and one of these reports were on the same screen, you should have to read both to tell them apart.

The sans-only typography is load-bearing. Inter at 17px does the body work; Inter at heavy weights (600/700/800) at large sizes does the display work. IBM Plex Mono handles code, eyebrows, marginalia, and pills. No serif — that's a different language (see stripe-blog). The palette stays cool: pure white, off-white, navy, purple. Warm amber appears only in the warn slot, by exception.

02

When to pick this

Pick this when

Technical product announcements. API launches. B2B SaaS feature pages. Press releases that engineers will read. Reports that want a fintech-grade trust signal without warmth or character.

Don't pick this when

Long-form essays where serif body is appropriate — use stripe-blog. Dense API reference where utility beats airiness — use stripe-docs. Warm internal long-form — use editorial-parchment.

03

Locked brand stack

Stripe ships their content in Söhne by Klim Type. Söhne is paid and not available on Google Fonts, so this language uses Inter as the realistic web fallback — geometric humanist sans, the closest free match for Söhne's letterforms. Document this fallback in writing if you ship outside Stripe-controlled surfaces.

<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;500;600;700;800&family=IBM+Plex+Mono:wght@400;500;600&display=swap" rel="stylesheet">

Voice rules

  • H1 carries an italic em phrase in purple: <h1>Title, <em>italic phrase</em></h1>.
  • Display sizes are large — clamp(48px,6.5vw,88px) on h1, 42px on h2. The press register reads at distance.
  • Body line-height stays at 1.65 — generous, never crowded.
  • Letter-spacing is tight on headings (−0.025em / −0.035em) — Inter only feels Stripe-quality with negative tracking.
  • Mono caps with letter-spacing for eyebrows and meta-row labels.
  • Never bold body prose for emphasis — use italic instead.
  • The eyebrow dot is always --purple — it is the brand's smallest purple touch.

Why amber for warn

Stripe's brand palette doesn't include an orange. A warm amber (#E5A53E) was chosen because (a) it reads as editorial-correct on a cool palette, (b) it is unambiguously a caution signal, and (c) it carries no fintech-money-loss association the way red would. The warn slot is intentionally small — it appears only inside .callout.warn and diagram warn nodes.

Warning: Do not introduce a third brand accent. Purple and navy carry the language; amber is reserved for caution. A success palette (green) is permissible but use sparingly — Stripe's voice is restrained, not celebratory.
05

Anti-patterns

  • No serif body. Inter is the body face. (Serif body belongs to stripe-blog.)
  • No warm backgrounds — no parchment, no cream, no beige. Surfaces are cool: pure white and off-white only.
  • No gradients on body content. Stripe's press-page hero is a flat white panel, not a gradient.
  • No dark mode for this register. The press language is light-only.
  • No saturated brand-clash colors (no electric green, no neon pink, no orange that isn't the amber warn).
  • No drop-shadows heavier than 0 2px 4px rgba(10, 37, 64, 0.03). The Stripe register is flat.
  • No emoji in headings or callouts. The voice is professional.
  • No marketing-page conventions — no "Get started" CTAs, no oversized button arrays, no testimonial cards. This is editorial.
  • No purple gradients. Purple is a flat accent. Stripe uses purple gradients on their marketing pages; the press register doesn't.

07

Diagram philosophy

Diagrams in this register are quiet and precise. Boxes have thin borders, rounded corners (rx="14"), white fills, and purple arrows. Text inside diagrams uses Inter — matching body — so the diagram and the surrounding prose stay in conversation.

Role-to-palette map

hotvar(--purple-soft)var(--purple)Same geometry; purple accent — primary focus.

SVG typography

Base text classes from diagram-kit — Inter for body-matching text:

.txt   { font: 14px "Inter", sans-serif; fill: var(--navy-deep); }
.small { font: 12px "Inter", sans-serif; fill: var(--text); }

Language extensions for annotation, step numbering, and alert text:

.note      { font: italic 12px "Inter", sans-serif; fill: var(--text-light); }
.step      { font: 500 11px "IBM Plex Mono", monospace; fill: var(--navy-deep); letter-spacing: 0.12em; }
.alert-txt { font: 600 11px "IBM Plex Mono", monospace; fill: var(--warn);       letter-spacing: 0.14em; }
LockedInline annotations use Inter italic in --text-light. Step numbers and alert text use IBM Plex Mono with letter-spacing — navy for steps, amber for alerts.
FreeWhether to use step numbers at all, exact letter-spacing (0.10–0.16em), annotation placement.

Alert region pattern

When a diagram has a failure path or exceptional region, wrap the affected messages in a warn-tinted dashed rectangle and place an inline alert pill (warn-soft fill + warn border + warn-colored mono caps text). The pattern is visible above between messages 03 and 04.

LockedWarn-tinted background uses rgba(229, 165, 62, 0.07) + dashed warn border. The inline alert pill stacks a colored dot, a mono-caps label, and an Inter italic detail.
FreeRegion size, exact alert pill copy, whether to include the colored dot.

Arrow color

Default arrows: var(--purple) at 2px. Return paths use arrow-muted (--rule-strong grey) so direction is legible at a glance. Async/fire-and-forget arrows add stroke-dasharray: 6 3.

Caption

Inter italic, --text-light, 14px, centered with margin-top: 16px.

Exemplar reports

Read at least one before writing a fresh report. Exemplars are evidence of the language in real content — not templates to clone.

  • ~/notes/04_Research/api-launch-announcement.html — canonical press-page report. Reference for "full Stripe Press at scale."
  • ~/notes/04_Research/release-notes-v8.html — shorter feature announcement. Same language at smaller scale.

The two exemplars are visibly different in composition but instantly recognizable as the same language. Locked brand stack, free composition.

If two reports in this language feel xeroxed, one of them is wrong. If two reports in this language feel like different brands, both of them are wrong.
