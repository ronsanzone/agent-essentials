Synthwave Outrun, set in neon

01

Personality

Outrun is the full-throttle synthwave. Deep night sky #0c0c1e as the base, magenta and cyan as the primary signal pair, an orange-to-yellow sun on the horizon for everything ceremonial. The display face is Audiowide — a chromed retro-futurist headline used only for the hero title and section headings. Body copy is Outfit, a clean geometric sans, because legibility still matters and Audiowide at 16px would be a war crime.

If the cover of an LP and the spread of a magazine had a child, Outrun is what they would name it.

Magenta #FF00FF is the lead accent — it does the work of links, pull quotes, active TOC items, and the primary diagram nodes. Cyan #00FFFF is the secondary — eyebrows, marginalia numbers, pills, success callouts. Orange-sun #FD8A26 stays in its lane as the warn color. The yellow-laser #FEF65B appears once, inside the hero <em>, like a sunset peeking through.

02

When to pick this

Pick this when

Editorial pieces on music, nostalgia, retro tech. Product launches with a deliberately maximalist 1980s register. Short-to-medium reports (~5–15 min) where the typography is part of the message. Anything that benefits from feeling like a poster.

Don't pick this when

The report is a 20+ minute operator reference. The audience is reading on the worst-contrast monitor in the office. The content is somber — incidents, postmortems, deprecations. Outrun is not a serious-document register.

Voice rules

  • Hero h1 in Audiowide uppercase with a text-shadow glow stack: magenta inner + cyan outer.
  • The <em> inside h1 is yellow-laser #FEF65B, not magenta — it is the one place yellow appears.
  • Pull quotes are large (≥26px), Outfit italic 500, magenta with a neon text-shadow.
  • Eyebrow is always mono caps cyan with a glowing cyan dot. Never magenta.
  • Never use Audiowide for h3 or smaller — at body sizes it becomes unreadable chrome soup.
  • Bold body prose for emphasis is acceptable here (unlike Editorial Parchment) — italic also works.

05

Anti-patterns

  • No Audiowide at body sizes. Audiowide is for h1 and h2 only.
  • No Arial, Roboto, or system-font defaults.
  • No light mode. Outrun is dark-only.
  • No more than one sunset gradient per document (the hero claims it; nothing else may).
  • No flat black #000 — the brand black is #0c0c1e with a violet bias.
  • No saturated emerald or pure red. Those are different palettes.
  • No drop shadows that don't glow. If a shadow doesn't 0 0 Npx in a neon hue, it doesn't belong here.
  • No emoji in section headings. Outrun is maximalist but not unserious.

07

Diagram philosophy

Diagrams in Outrun are album covers. A sunset gradient sky bleeds purple-to-orange behind every figure, a perspective grid races toward a vanishing point on the horizon, and boxes ride the grid with chrome bevel and neon glow. Headlines wear Audiowide; the rest of the body type stays Outfit so the prose and the diagram still belong to the same conversation. hot nodes carry the magenta-to-cyan chrome gradient so the focus reads from across the room.

SVG typography

Headlines inside diagrams use Audiowide to match the section h2; body labels stay in Outfit. SVG text gets its glow from filter="url(#so-glow)" rather than CSS text-shadow, since SVG renderers ignore the latter.

.so-txt   { font: 400 14px "Audiowide", sans-serif; fill: #fdf6ff; letter-spacing: 0.06em; }
.so-small { font: 12px "Outfit", sans-serif; fill: #f0d8ff; }

.so2-note  { font: italic 12px "Outfit", sans-serif; fill: #c0a8e6; }

Alert region pattern

When a diagram has a failure path, wrap the affected messages in an orange-sun dashed rectangle with a partial sun disc peeking up from below (clipped by the region so only the upper crescent shows — the sun is literally setting on this code path). Inline alert pill carries an orange-sun glowing dot, Audiowide caps label, and an Outfit italic detail. The pattern is visible above between messages 03 and 04.

Arrow color

Default arrows are magenta #FF00FF at 2px and always pass through the neon-glow filter. Return paths use arrow-muted (muted violet, no glow) so direction is legible. Async dispatch dashes the magenta line (stroke-dasharray: 6 3) and keeps its glow.
